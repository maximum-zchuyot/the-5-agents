# יעל — סוכנת כתיבת תוכן

מסמך מצביע לבני אדם. הגדרת הסוכן הקנונית נמצאת ב-`.claude/agents/yael.md`.

---

## תפקיד

יעל משכתבת מאמרי גלם (מ-`Content/`) בסגנון הכתיבה שלנו, כפי שמוגדר ב-`yael/style-guide.md`
ומודגם בדוגמאות ב-`yael/reference/`.

---

## הפעלה

ראובן מפעיל את יעל דרך ה-Agent tool כאשר מתקבלת בקשה לשכתוב/עריכת תוכן.
יעל מעבדת **מאמר אחד בלבד** לכל הפעלה — ראובן מציין את שם הקובץ.

---

## קבצים מרכזיים

| קובץ/תיקייה | תפקיד |
|---|---|
| `yael/style-guide.md` | מדריך הסגנון — **מלאו אותו לפני שמשתמשים ביעל** |
| `yael/reference/` | דוגמאות לטקסטים בסגנון שלנו — הוסיפו קבצי `.md` |
| `Content/` | תיבת דואר נכנס — זרקו כאן מאמרי גלם לעיבוד |
| `Content/Ready/` | ארכיון — מאמרי המקור לאחר שיעל עיבדה אותם |
| `Output/` | פלט — המאמרים המשוכתבים מופיעים כאן |

---

## פורמט IMAGE_NEEDED

כשיעל מזהה שתמונה תחזק את המאמר, היא מוסיפה placeholder:

```
{{IMAGE_NEEDED: "<תיאור מפורט של התמונה הרצויה>"}}
```

ראובן קורא לפלט, מוצא את ה-placeholders, מפעיל את **יובל** עם כל תיאור,
ומחליף את ה-placeholder ב-markdown image reference לנתיב שיובל החזיר.

---

## כלים מותרים

Read, Write, Edit, Glob, Grep בלבד.
יעל לא יכולה לגלוש באינטרנט, ליצור תמונות, לגשת ל-API, או להפעיל סוכנים אחרים.
