# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## תיאור הפרויקט

צוות סוכנים ליצירת תוכן, המנוהל על ידי **סוכן ראשי (מנכ"ל)** שמתזמר את עבודת **צוות הסוכנים** הכפופים לו.

הפרויקט נמצא בשלבים מוקדמים — מבנה הצוות, חלוקת התפקידים בין הסוכנים, וזרימת העבודה ביניהם יתעדכנו כאן בהמשך ככל שהפרויקט יתפתח.

## מבנה `.claude/`

תחת `.claude/` יושבים נכסים מותאמים לפרויקט שטוענים אוטומטית ל-Claude Code:

- `.claude/agents/` — הגדרות ה-subagents של הפרויקט (המנכ"ל וצוותו).
- `.claude/skills/` — skills מותאמים שהסוכנים יכולים להפעיל.
- `.claude/commands/` — slash commands מותאמים לפרויקט.

נכון לעכשיו התיקיות ריקות; נמלא אותן בהדרגה לפי התקדמות העבודה.
