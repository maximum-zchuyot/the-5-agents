# Yuval — Agent Pointer

This directory is Yuval's working space. The canonical **Claude agent definition** lives at:

→ `.claude/agents/yuval.md`

Do not edit this file to change agent behavior — edit the canonical file above.

---

## Directory Layout

| Path | Purpose |
|---|---|
| `reference/` | Drop reference images here. Yuval reads them before every generation to extract style cues. |
| `outputs/` | All generated images land here as `<YYYY-MM-DD>-<slug>.png`, each with a sibling `.txt` file containing the exact prompt used. |
| `agent.md` | This file — pointer doc for humans. |
| `skill.md` | Pointer to the image-generation skill Yuval delegates to. |

---

## How to use Yuval

1. **Add reference images** to `reference/` — any PNG/JPG that represents the desired visual style.
2. **Ask Yuval** (via the CEO agent) to generate an image. Example prompts:
   - "תמונה של hero banner לאתר בסגנון מינימליסטי"
   - "generate a product shot for the new campaign"
3. **Iterate** — edit the `.txt` prompt file in `outputs/` and ask Yuval to regenerate.

Yuval uses reference images automatically; no extra instructions needed.
